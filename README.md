# Admin Panel - React (Frontend-only)

This is a single-page React admin panel demo for employee management, attendance tracking and salary preview.
**Features**
- Employees CRUD (stored in localStorage)
- Mark attendance per date
- Salary preview per month (simple calculation)
- Pages: Dashboard, Employees, Attendance, Salaries, About

**How to run**
1. Install Node.js (v18+ recommended)
2. In project folder:
   - `npm install`
   - `npm run dev`
3. Open the URL shown by Vite (usually http://localhost:5173)

**Notes**
- This is frontend-only. For production, connect to a backend and a real database.
- Data is stored in browser localStorage under key `adminpanel_employees_v1`. To reset, clear localStorage or open devtools -> Application -> Local Storage.

Enjoy!
